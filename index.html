Feedory

Self-hosted personal feed memory. It works as a buffering online adapter for your local feed subscriptions (not an online news reader)

View project on GitHub

Features

  • supports any number of newsfeed subscriptions
  • keeps the news as long as you want
  • updates its newsitem database regularly, supported by your host's cron daemon or an external webbased cron service
  • reads all usual kinds of newsfeed formats
  • imports opml files and exports opml subscriptions lists with the feedory rss url or the original source rss url
  • has a webbased administration interface for handling your subscriptions efficiently
  • supports autodetection of feeds on web page url
  • supports subscriptions on the fly while you surf the net via bookmarklet
